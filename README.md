# 📊 Amazon Product Analysis Dashboard (Excel)

## 🔍 Overview

This project presents an interactive Excel dashboard analyzing Amazon product data across multiple categories. It highlights pricing trends, customer satisfaction, and engagement using PivotTables and visualizations.

## 📌 Key Features

* Interactive slicer for filtering categories
* Average Price analysis by category
* Customer Satisfaction (Ratings) comparison
* Total Reviews analysis for engagement insights
* KPI metrics for quick business understanding

## 📈 Tools Used

* Microsoft Excel
* PivotTables
* PivotCharts
* Slicers
* Data Cleaning

## 💡 Key Insights

* Electronics category has the highest average price and review volume
* Office Products and Toys & Games show higher customer satisfaction ratings
* Some categories have lower engagement, indicating growth opportunities
